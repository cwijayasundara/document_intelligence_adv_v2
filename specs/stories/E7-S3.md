# E7-S3: Upload Page with Drag-Drop

**Epic:** E7 — Frontend Core
**Layer:** UI
**Group:** C
**Depends on:** E7-S1, E2-S1

## Description

File upload page with drag-drop zone using react-dropzone. Supports PDF, DOCX, XLSX, and image formats. Shows upload progress and handles duplicate detection from API response.

## Acceptance Criteria

1. Upload page at /upload renders a drag-drop zone using react-dropzone
2. Accepts file types: PDF (.pdf), Word (.docx), Excel (.xlsx), images (.png, .jpg, .tiff) with visual file-type icons
3. Upload progress indicator shown per file during the upload API call
4. Duplicate file response from API (existing document returned) shows notification: "File already exists" with link to existing document
5. Successful upload navigates to /documents/:id/parse for the newly created document
