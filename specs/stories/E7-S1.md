# E7-S1: Frontend App Shell, Routing, API Client

**Epic:** E7 — Frontend Core
**Layer:** UI
**Group:** B
**Depends on:** —

## Description

React 19 + Vite + TypeScript project scaffold with Tailwind CSS, React Router, Axios API client, and TanStack Query. Establishes the design token system (colors, typography, shadows) adapted from the reference frontend.

## Acceptance Criteria

1. React 19 + Vite + TypeScript project created in frontend/ with Tailwind CSS 4 configured
2. React Router configured with all routes: /, /upload, /documents/:id/parse, /documents/:id/classify, /documents/:id/extract, /documents/:id/summary, /documents/:id/chat, /config/categories, /config/extraction-fields, /bulk
3. Axios API client configured with baseURL http://localhost:8000/api/v1 and snake_case↔camelCase request/response transformers
4. TanStack Query provider configured with staleTime=5min, no retry on 404/401/403, max 2 retries with exponential backoff
5. Design token system in tailwind.config.ts: primary (blue), success (green), warning (amber), error (red) color palettes, Inter + Poppins fonts, shadow scale, border-radius scale
