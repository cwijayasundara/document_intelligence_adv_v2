# E8-S5: RAG Chat Page with Citations

**Epic:** E8 — Frontend AI & Config
**Layer:** UI
**Group:** E
**Depends on:** E7-S1, E6-S3

## Description

Chat interface for RAG-based document querying. Supports scope selection (single doc, all, by category), search mode toggle, and displays answers with expandable citation cards.

## Acceptance Criteria

1. Chat page at /documents/:id/chat renders a message-style conversation interface with input field
2. Scope selector: "This Document" (default, uses document_id), "All Documents", "By Category" (shows category dropdown)
3. Search mode toggle: Semantic, Keyword, Hybrid (default) — maps to search_mode parameter
4. Query submission sends POST /api/v1/rag/query; response rendered as AI message with expandable citation cards showing chunk_text, document_name, relevance_score
5. Chat history maintained in component state for multi-turn conversation display within the session
