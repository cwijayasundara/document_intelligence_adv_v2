# E3-S2: PII Filtering Middleware

**Epic:** E3 — Agent Framework & Memory
**Layer:** Service
**Group:** C
**Depends on:** E3-S1

## Description

DeepAgents middleware that filters PII from document content before it reaches the LLM. Registered as a pre-model callback. Redacts investor names, SSN/tax IDs, addresses, bank account details, phone numbers. Passes through fund names and financial terms unredacted.

## Acceptance Criteria

1. PII middleware registered as a DeepAgents pre-model callback that intercepts content before LLM calls
2. Regex patterns detect and replace with [REDACTED_PII]: SSN/tax IDs (XXX-XX-XXXX), phone numbers, email addresses, bank account/routing numbers, US street addresses
3. Fund names and financial terms (management fee, carried interest, preferred return, fund term, commitment period) pass through unredacted
4. [REDACTED_PII] placeholders include a type hint: [REDACTED_SSN], [REDACTED_PHONE], [REDACTED_EMAIL], [REDACTED_ADDRESS], [REDACTED_ACCOUNT]
5. Original unredacted content is never passed to the LLM — only the redacted version
