# E9-S3: Bulk Upload + Dashboard UI

**Epic:** E9 — Bulk Processing
**Layer:** UI
**Group:** F
**Depends on:** E9-S2, E7-S1

## Description

Bulk processing dashboard showing all jobs with progress bars and per-document status. Includes multi-file upload interface for starting new bulk jobs.

## Acceptance Criteria

1. Bulk dashboard at /bulk displays all bulk jobs in a list with status badge, progress bar (processed_count / total_documents), and timestamps
2. "New Bulk Job" section with multi-file drag-drop upload zone (same file types as single upload)
3. Per-document progress view: expanding a job row shows each document with status indicator (pending=gray, processing=blue, completed=green, failed=red)
4. Failed documents show error_message text with visual error styling
5. Job list auto-refreshes via TanStack Query with 5-second polling interval while any job has status=processing
