# E1-S3: Database Connection & Migrations

**Epic:** E1 — Foundation Infrastructure
**Layer:** Repository
**Group:** A
**Depends on:** —

## Description

Async database connection layer using SQLAlchemy 2.0 with asyncpg driver. Includes session factory with connection pooling and Alembic configured for async migrations. Initial migration creates all tables.

## Acceptance Criteria

1. Async SQLAlchemy engine created with asyncpg driver using DATABASE_URL from config
2. Async session factory configured with connection pooling (pool_size=5, max_overflow=10)
3. Alembic configured for async migrations with proper env.py setup
4. Initial migration creates all 10 tables (8 core + 2 memory) matching ORM models
5. `alembic upgrade head` runs successfully against an empty PostgreSQL 16 database
