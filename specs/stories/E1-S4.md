# E1-S4: FastAPI Application Factory + Health Endpoint

**Epic:** E1 — Foundation Infrastructure
**Layer:** API
**Group:** A
**Depends on:** —

## Description

FastAPI application factory with CORS middleware, lifespan handler for database pool management, and a health endpoint that verifies database connectivity.

## Acceptance Criteria

1. FastAPI app created via factory function with CORS middleware allowing localhost origins
2. GET /api/v1/health returns 200 with {"status": "healthy"} when database is reachable
3. Health endpoint returns 503 with {"status": "unhealthy", "detail": "..."} when database is unreachable
4. All API routes use /api/v1 prefix
5. Lifespan handler initializes database connection pool on startup and closes on shutdown
