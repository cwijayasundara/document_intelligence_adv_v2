# E8-S4: Summary Page with Regenerate

**Epic:** E8 — Frontend AI & Config
**Layer:** UI
**Group:** D
**Depends on:** E7-S1, E6-S1

## Description

Document summary page showing the generated summary text and key topics. Supports generation, regeneration, and progression to ingestion.

## Acceptance Criteria

1. Summary page at /documents/:id/summary displays the summary text and key topics as tags
2. "Generate Summary" button triggers POST /api/v1/summarize/:id with loading state
3. "Regenerate" button re-triggers summarization, replaces displayed summary with new result
4. Key topics rendered as styled tag chips below the summary text
5. "Proceed to Ingest & Chat" button navigates to /documents/:id/chat after triggering POST /api/v1/ingest/:id
