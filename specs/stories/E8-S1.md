# E8-S1: Config Management Pages

**Epic:** E8 — Frontend AI & Config
**Layer:** UI
**Group:** C
**Depends on:** E7-S1, E4-S1

## Description

Admin pages for managing document categories and extraction field schemas. Category page supports CRUD with classification criteria editor. Field editor supports add/edit/reorder with drag-drop.

## Acceptance Criteria

1. Category manager at /config/categories displays all categories in a card list with name, description, and edit/delete actions
2. Add/edit category modal includes fields: name (required), description, classification_criteria (textarea)
3. Delete category shows confirmation dialog; disabled if documents are assigned to the category
4. Extraction field editor at /config/extraction-fields shows fields grouped by category with drag-drop reordering for sort_order
5. Add/edit field form includes: field_name, display_name, description, examples (textarea), data_type (dropdown: string/number/date/currency/percentage), required (checkbox)
