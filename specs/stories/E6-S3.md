# E6-S3: RAG Retriever Subagent + Query API

**Epic:** E6 — Summarization & RAG
**Layer:** Service + API
**Group:** D
**Depends on:** E3-S1, E6-S2, E1-S4

## Description

DeepAgent subagent for RAG-based document retrieval using Weaviate hybrid search. API endpoint accepts queries with scope filtering (single doc, all, by category) and search mode selection (semantic, keyword, hybrid).

## Acceptance Criteria

1. RAG retriever registered as DeepAgent subagent with weaviate_hybrid_search tool
2. Hybrid search supports alpha parameter: 0.0 = pure keyword (BM25), 0.5 = balanced (default), 1.0 = pure semantic (vector)
3. POST /api/v1/rag/query accepts: query (str), scope (single_document|all|by_category), scope_id (document_id or category_id), search_mode (semantic|keyword|hybrid), top_k (int, default 5)
4. Response includes: answer (str), citations (list of {chunk_text, document_name, document_id, relevance_score})
5. Scope filters applied as Weaviate where-filters: single_document filters on document_id, by_category filters on document_category
