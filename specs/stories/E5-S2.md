# E5-S2: Judge Subagent with Confidence Scoring

**Epic:** E5 — Extraction & Confidence Judging
**Layer:** Service
**Group:** C
**Depends on:** E3-S1, E3-S2

## Description

Separate DeepAgent subagent that evaluates extraction quality. Receives extracted values + source texts + original document, returns confidence rating (high/medium/low) with reasoning per field. Separate LLM call ensures objectivity.

## Acceptance Criteria

1. Judge registered as a separate DeepAgent subagent (independent from extractor) with tools: get_extracted_values, get_parsed_content
2. Judge evaluates each extracted field against its source_text and the full document context
3. Returns JudgeResult with list of FieldEvaluation(field_name, confidence: "high"|"medium"|"low", reasoning: str)
4. Confidence criteria enforced: high = source text explicitly states value; medium = value implied/requires interpretation; low = vague, contradictory, or no clear source
5. PII middleware applied before the judge LLM call
