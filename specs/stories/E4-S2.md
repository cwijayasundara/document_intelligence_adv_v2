# E4-S2: Classifier Subagent

**Epic:** E4 — Classification & Config Management
**Layer:** Service
**Group:** C
**Depends on:** E3-S1, E3-S2, E4-S1

## Description

DeepAgent subagent that classifies parsed documents against user-defined categories. Receives parsed markdown + all categories with criteria. Returns the best-matching category with reasoning. PII middleware applied.

## Acceptance Criteria

1. Classifier registered as a DeepAgent subagent with tools: get_categories (loads all categories from DB), get_parsed_content (reads parsed markdown)
2. Classifier returns ClassificationResult structured output: category_id (UUID), category_name (str), reasoning (str)
3. PII middleware filters content before the classification LLM call
4. Classification validated: returned category_id must exist in the database
5. Returns "Other/Unclassified" category if no category matches with sufficient confidence
