# E6-S1: Summarizer Subagent + API

**Epic:** E6 — Summarization & RAG
**Layer:** Service + API
**Group:** C
**Depends on:** E3-S1, E3-S2, E1-S4

## Description

DeepAgent subagent that generates document summaries with key topics. API endpoint triggers summarization, stores results with content hash for cache invalidation, and supports regeneration.

## Acceptance Criteria

1. Summarizer registered as DeepAgent subagent with get_parsed_content tool, returns SummaryResult(summary: str, key_topics: list[str])
2. PII middleware applied before the summarization LLM call
3. Summary saved to document_summaries table with content_hash (SHA-256 of parsed content)
4. POST /api/v1/summarize/:id generates summary, transitions document status to summarized, returns summary + key_topics
5. GET /api/v1/summarize/:id returns existing summary; returns 404 if no summary exists
6. POST /api/v1/summarize/:id with existing summary regenerates if parsed content hash differs, otherwise returns cached summary
